# Public Assets

## Resume/CV

Please place your resume/CV PDF file in this directory with the filename `resume.pdf` for the download functionality to work properly.

The resume download button is available in:
- Navbar (desktop & mobile)
- Hero section

**Expected file:** `resume.pdf`
**Location:** `/public/resume.pdf`

Once placed, it will be accessible at: `https://yourdomain.com/resume.pdf`
